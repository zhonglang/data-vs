# 城市数据对比视频生成器 (City Data Comparison Video Generator)

这是一个基于 Web 技术的视频生成工具，专为制作数据对比类短视频（如“各省市 GDP 大比拼”）而设计。用户可以通过简单的 JSON 数据输入，生成动态的、带有视觉特效的对比视频，并直接导出为 MP4/WebM 格式，方便在抖音、B站等自媒体平台发布。

## 主要功能

### 1. 数据可视化与配置
*   **JSON 数据导入**: 支持自定义 JSON 格式数据输入，自动解析分类（如省份）和条目（如城市及数值）。
*   **灵活对比**: 支持任意选择两个分类进行左右侧数据对比（Left vs Right）。
*   **自定义外观**:
    *   可修改主标题和副标题。
    *   支持自定义背景渐变色（上/下颜色配置）。

### 2. 动态演示效果
*   **排名展示**: 自动计算并显示条目排名。
*   **视觉特效**:
    *   炫光标题与动态背景扫描效果。
    *   条目进场动画与数值进度条动画。
    *   胜负高亮显示（数值高的一方会有特殊光效）。
*   **自动滚动**: 支持智能滚动模式，确保最新展示的数据始终在可视区域。

### 3. 视频导出
*   **高清录制**: 使用 Canvas + MediaRecorder 技术，将动态效果逐帧渲染并录制。
*   **独立渲染**: 导出过程使用独立的 Canvas 渲染，不受浏览器窗口大小影响，确保输出标准的 9:16 竖屏视频（适配手机端）。
*   **格式支持**: 根据浏览器支持情况，自动选择导出 MP4 (H.264) 或 WebM 格式。

## 使用说明
1.  **打开项目**: 直接在浏览器中打开 `index.html` 文件。
2.  **录入数据**: 在“配置面板”中粘贴符合格式的 JSON 数据，点击“解析数据”。
3.  **配置参数**: 选择对比双方，调整标题和配色。
4.  **预览与导出**:
    *   点击“开始播放”在网页右侧预览动画效果。
    *   点击“导出视频”开始后台渲染并下载视频文件。

## 技术栈
*   **核心**: HTML5, CSS3 (Flex/Grid, Animation, Backdrop Filter), Vanilla JavaScript (ES6+).
*   **图形与媒体**: HTML5 Canvas API (用于视频帧绘制), MediaStream Recording API (用于视频流捕获与保存).
